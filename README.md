# ESP32 Метеостанція + Вебрадіо

Проєкт на базі ESP32: датчик **BME280** (температура, вологість, атмосферний тиск) та **вебрадіо** (інтернет-радіо) з 5 попередньо заданими станціями, аудіовихід через I2S-підсилювач MAX98357A.

## Компоненти

| Компонент | Призначення |
|---|---|
| ESP32 (будь-яка dev-плата) | Мікроконтролер |
| BME280 | Температура / вологість / тиск |
| MAX98357A | I2S DAC + підсилювач класу D |
| Динамік 4-8 Ом | Аудіовихід |
| Кнопка | Перемикання станцій |

## Схема підключення

**BME280 (I2C):**
- VCC → 3.3V
- GND → GND
- SDA → GPIO21
- SCL → GPIO22
- Адреса: `0x76` (якщо не працює — спробуйте `0x77`)

**MAX98357A (I2S):**
- VIN → 5V
- GND → GND
- BCLK → GPIO27
- LRC → GPIO26
- DIN → GPIO25

**Кнопка:**
- Один контакт → GPIO4
- Другий контакт → GND
- (використовується внутрішній `INPUT_PULLUP`, зовнішній резистор не потрібен)

## Бібліотеки

Встановіть через Arduino IDE Library Manager:
- `Adafruit BME280 Library`
- `Adafruit Unified Sensor`
- [`ESP32-audioI2S`](https://github.com/schreibfaul1/ESP32-audioI2S) (автор schreibfaul1)

## Налаштування

1. Відкрийте `src/esp32_bme280_webradio.ino` в Arduino IDE.
2. Впишіть дані вашого Wi-Fi у змінні `WIFI_SSID` та `WIFI_PASSWORD`.
3. За бажання замініть URL-адреси у масиві `stations[]` на власні (шукати потокові посилання можна на [radio-browser.info](https://www.radio-browser.info)).
4. Завантажте скетч на плату (обрати плату: ESP32 Dev Module).

## Використання

- Кнопка на GPIO4 перемикає між 5 станціями по колу.
- Дані з BME280 виводяться в Serial Monitor (115200 бод) кожні 30 секунд.
